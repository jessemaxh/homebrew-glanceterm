# homebrew-glanceterm

Homebrew tap for [GlanceTerm](https://github.com/jessemaxh/GlanceTerm) — a terminal with a sidebar for managing many AI coding agents at once.

## Install

```sh
brew install --cask jessemaxh/glanceterm/glanceterm
```

or:

```sh
brew tap jessemaxh/glanceterm
brew install --cask glanceterm
```

Apple Silicon (arm64) only for now. The cask version is kept in sync with the
latest stable GlanceTerm release automatically by `.github/workflows/update-cask.yml`.

> Currently pinned to a pre-release test build until the first stable
> GlanceTerm release ships.
